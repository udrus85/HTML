# Mini HTML Project

Небольшой статический сайт для портфолио.

Содержит:
- `index.html` — страница
- `styles.css` — простые стили
- `demo.html` — демонстрационная страница с небольшим примером

Как запустить: откройте `index.html` или `demo.html` в браузере.

История коммитов (пример):
1. chore: init html-mini-project with index.html — инициализация проекта
2. feat: add contact section to index.html — добавлена секция контактов
3. docs: add README and LICENSE — обновлён README и добавлена лицензия
4. feat: add demo page — добавлена demo.html (feature branch)
